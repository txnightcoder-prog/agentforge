How AgentForge Works
Step 1
Query Received
User submits AI query to Token Metering Engine.
Step 2
Complexity Classification
Engine classifies query against three-tier schema.
Step 3
Entitlement Validation
Checks purchased tier against classified complexity.
Step 4
Gate or Allow
Within tier → approved. Exceeds tier → gated, user prompted to upgrade.
Step 5
Tier Routing
Routed to compute path: lightweight, standard LLM, or full agentic pipeline.
Step 6
Ledger Recording
Logs: user ID, tier, timestamp, compute path, remaining balance.
